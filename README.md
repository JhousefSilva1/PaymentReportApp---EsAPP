#  Payment API

API REST desarrollada en **.NET 8** para el registro y consulta de pagos de servicios básicos (agua, electricidad, telecomunicaciones).  
Este proyecto forma parte de una prueba técnica y está construido con buenas prácticas de arquitectura y validaciones de negocio.

---

##  Tecnologías Utilizadas
- **.NET 8 (ASP.NET Core Web API)**
- **Entity Framework Core (InMemory)**
- **Swagger / OpenAPI** (para documentación interactiva)
- **Visual Studio 2022 / VS Code / Visual Studio 2022 2026

---

##  Requisitos Previos
- SDK de **.NET 8.0 o superior**
- (Opcional) **Visual Studio 2022** o **VS Code**
- (Opcional) **Postman** o navegador para probar endpoints

---

## Archvo global.json
- Este archivo asegura que el proyecto utilice la versión específica del SDK de .NET 8.0.415
- Si tienes múltiples versiones del SDK instaladas, este archivo ayuda a evitar conflictos y garantiza que el proyecto se ejecute con la versión correcta.
